## Quick Answers

### Getting Started

**Q: How do I deploy my first app?**
A: Follow the [Quickstart Guide](/docs/getting-started/QUICKSTART) - you can deploy in under 5 minutes.

**Q: Do I need Docker knowledge?**
A: No. Enclii supports Buildpacks for automatic containerization, but you can use Dockerfiles if you prefer.

**Q: What's the minimum to get started?**
A: A GitHub account and your code. That's it.

### Platform Basics

**Q: Where does my code run?**
A: On dedicated Hetzner infrastructure managed by the MADFAM team, with Cloudflare for edge security.

**Q: Is there a free tier?**
A: Enclii is designed for organizations. Contact us for pricing based on your needs.

**Q: Can I use my own domain?**
A: Yes. Custom domains with automatic SSL are fully supported.

### Technical

**Q: What regions are supported?**
A: Currently, we run on European infrastructure (Germany). Additional regions available upon request.

**Q: What's the uptime SLA?**
A: 99.5% for standard services, with 99.95% available for critical workloads.

**Q: Can I run background jobs?**
A: Yes. Enclii supports cron jobs and one-off tasks via the Timetable subsystem.
